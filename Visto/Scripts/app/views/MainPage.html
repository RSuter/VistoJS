This is the Visto JavaScript Library sample application.

The look and feel of this sample application is only for demonstration purposes and can completely be changed.
Edit the CSS file in the "css" directory and replace the images in the "images" directory.
Only files from the "libs" and "common" directory are needed for the Visto JavaScript Library.

Navigation & paging

  • Navigate to tab page: A sample of a tabbed page
  • Navigate to list page: A sample of list and details page
  • Navigate to web view page: A sample of page with a web view
(The navigation frame is the HTML's body element)

Internationalization

The translated strings can be found in the "languages" directory of each package.

Bindings

Demonstration of a two-way binding into a sub view (SampleControl).
The text in the input box is bound to the text of the SampleControl below.
